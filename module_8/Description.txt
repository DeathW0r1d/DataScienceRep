min MAPE: 10.68121
Kaggle: https://www.kaggle.com/mikedenisov/baseline-sf-dst-car-price-part2-v1
Описание: 
Проект является логическим продолжением предыдущих проектов в этой области и требует построение многовходовой нейросети, обрабатывающей:
	- Табличные данные из объявлений о продаже автомобиля;
	- Текстовые данные из описаний;
	- Изображения, прикрепленные к объявлению.
Требуется:
1. Построить наивную ML - модель на табличных данных для формирование "базового" уровня рассматриваемой метрики;
2. Первую ML - модель на основе CatBoost;
3. DL - модель на основе табличной нейросети.
4. Добавить обработку естественного языка и формирование Multi-Input (NLP+TNN) NN;
5. Добавить обработку изображений в Multi-Input (NLP+IP+TNN) NN;
6. Произвести ансамблирование моделей.

Итоги: 
1. Построены модели, при этом для обработки изображений использовались облегченные SotA - модели по типу EfficientNetB3 для извлечения признаков, предобученные на
ImageNet. 
2. Для NLP на данный момент реализована классическая рекуррентная сеть (правда, с LSTM - элементами, поэтому проблемы забывания/запоминания больше не являются такими уж
острыми)
3. Признаки обработаны и извлечены, из них сформирована сеть, исследована влияние логарифмирования и нормализации числовых признаков на итоговую метрику MAPE. 
4. Применено ансамблирование в виде простого оптимизационного алгоритма, подбирающего коэффициенты с использованием метода Нелдера-Мида (не требует формирования первой
производной для поиска решения).

Вывод: в рамках представленной работы наиболее эффективной как с точки зрения MAPE, так и с точки зрения затрат по обучению себя показала модель 
Nelder-Mead(CatBoostRegressor + TNN) - MAPE 10.68121
